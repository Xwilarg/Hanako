# Hanako
Hanako is a Robot Vacuum Cleaner<br/>
There is still a long way before it can clean anything, but I'm working on it

## Materiel
 - Arduino ATmega2560
 - 28BYJ-48 Stepper Motor x2
 - ULN2003 Driver Board Carte Pilot x2
 - Green LED
 - KY-006 Passive Buzzer
 - Wires
